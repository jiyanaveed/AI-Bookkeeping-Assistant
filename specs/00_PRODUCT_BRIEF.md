# Product Brief — UK AI Accountant Agent

## Product Name
UK AI Accountant Agent

## Goal
Build a UK-focused AI bookkeeping, finance-ops, and compliance assistant for:
- sole traders
- freelancers
- landlords
- micro-businesses
- limited companies

## Core Promise
Help users stay organized, deadline-aware, and accountant-ready.

## What the product does
- chat with users in plain English
- collect financial and company information through chat
- scan receipts and documents
- classify and summarize transactions
- calculate bookkeeping summaries
- track Companies House deadlines
- create reminders for upcoming deadlines
- explain financial and compliance information clearly
- keep outputs grounded in tools and structured data

## What the product does NOT do
- it is not a chartered accountant
- it is not a tax adviser
- it is not a lawyer
- it does not guarantee legal or tax compliance
- it must never fabricate figures, deadlines, or filing status
- it must never silently mutate records

## Phase 1 Scope
Only build the first reliable workflow:

1. user asks about a UK company
2. system finds the company
3. system fetches profile and important deadlines
4. system explains the deadlines
5. system can create reminders after explicit confirmation

## Phase 2 Scope
- receipt scanning
- transaction categorization
- monthly financial summaries
- basic bookkeeping assistant flows

## Phase 3 Scope
- WhatsApp integration
- Slack integration
- accountant collaboration mode
- document vault
- anomaly detection

## Primary Success Criteria
- no hallucinated deadline answers
- no invented company details
- conservative and trustworthy responses
- clear distinction between verified data and inferred data
- reminder creation only with explicit confirmation
